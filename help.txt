cross-check:

cross-check is designed to generate a list of addition, subtraction, multiplication, 
and devision, equations who's solution is a user designated number.

to set this number set EQUATEQ to your chosen number and un-comment the table size 
you wish to use, make sure that only one of the tables is un-commented.

vartable:

vartable is designed to solve simnd ple variable equations.
such as x+2=4 6-x=2.

it can handle addition, subtraction, division, and multiplication.

vartable-basenum.sh can even calculate in base number other than 10.

this is done by setting "base" as a number from 2-10, including 2 and 10.

repititionlimit:

this is a very complicated script used for generating single column, repitition calculations tables.

see the commented help in the script for more information.
